== Description ==

Restoneu-Ex  is a beautiful restaurant theme that provides more customizing options to restaurant people or freelancers to create a good website for company. restoneu comes with some exciting features to create a awesome website. It has the customizing options like blog style, logo update, layout control, color modification, extra widgets, sticky menu and much more.
